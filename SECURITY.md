# Security Policy

## Supported Versions
This project is in active development. Security fixes are applied to the latest default branch.

## Reporting a Vulnerability
If you discover a security issue, please do **not** open a public issue.

Instead:
1. Report privately via GitHub Security Advisories (preferred), or
2. Contact the maintainer directly through repository contact channels.

Please include:
- Description of the issue
- Reproduction steps / proof of concept
- Potential impact
- Suggested remediation (if known)

## Response Targets
- Initial acknowledgment: within **72 hours**
- Triage/update: within **7 days**
- Fix timeline: depends on severity and complexity

Thank you for helping keep this project and community safe.
